# gsc-archive

## How are these downloaded?
Wouldn't you like to know? I'm joking, they're downloaded using `scripts/download-gsc-images.sh`

## What is [blah blah blah chip name]?
Refer to this:
- `Cr50`: Firmware running on the H1B{2,3}{P,C} (Haven) chips
- `Ti50`: Firmware running on the H1D3C (Dauntless) chips or the new OpenTitan-based chips called NuvoTitan (NT) 
- `nugget-os`: Firmware running on the Pixel-specific GSCs. (e.g: Citadel & Dauntless)
